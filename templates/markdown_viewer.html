⚠️ Accessibility Issues

Markdown Cheat Sheet

Headers
# H1 Header ## H2 Header ### H3 Header #### H4 Header ##### H5 Header ###### H6 Header
Emphasis
*italic* or _italic_ **bold** or __bold__ ***bold italic*** ~~strikethrough~~
Lists
- Unordered item
* Another item
+ Plus works too 1. Ordered item
2. Another item
Task Lists
- [ ] Unchecked task
- [x] Checked task
- [ ] Another task
Links & Images
[Link text](url) [Link](url "Title") ![Alt text](image.jpg) <https://auto-link.com>
Code
`inline code` ```javascript
code block
```
Quotes & Lines
> Blockquote text
>> Nested quote --- (Horizontal rule)
Tables
| Header 1 | Header 2 |
|----------|----------|
| Cell 1 | Cell 2 |
Escaping & Special
\* Escape with backslash \# Not a heading <kbd>Ctrl</kbd> (HTML)
Accessibility Tips
  • Always provide alt text for images
  • Use proper heading hierarchy (H1→H2→H3)
  • Write descriptive link text (avoid "click here")
  • Use tables for tabular data only
  • Add language identifiers to code blocks
  • Keep paragraphs concise and scannable
